# 🚀 Logic Flow Mapper — 설치하고 사용하기

처음 보는 사람도 따라할 수 있게 만든 가이드입니다. 그냥 순서대로 따라하세요.

---

## 📌 처음 한 번만 설치 (총 5~10분)

### 1단계 — PowerShell 열기 (⚠️ cmd 아님!)

**방법 A (추천)** — `Windows 키 + S` (검색) → 검색창에 **"PowerShell"** 입력 → **Windows PowerShell** 클릭

**방법 B** — `Windows 키 + X` → 메뉴에서 **터미널** 또는 **Windows PowerShell** 클릭

> ⚠️ "명령 프롬프트" 또는 "cmd" 가 아닌 **PowerShell** 이어야 합니다.
>
> 확인 방법: 프롬프트가 **`PS C:\Users\...>`** 처럼 **`PS`** 로 시작하면 PowerShell 맞습니다.
>
> 만약 그냥 `C:\Users\...>` 라면 cmd 입니다. 그 창에 `powershell` 입력 후 Enter 누르면 전환됩니다.

### 1.5단계 — 실행 정책 풀기 (최초 1회만)

PowerShell 창에 아래 명령어 붙여넣고 Enter:

```powershell
Set-ExecutionPolicy -Scope CurrentUser -ExecutionPolicy RemoteSigned -Force
```

→ 메시지 없이 끝나면 성공.

> ⚠️ Windows 가 기본으로 PowerShell 스크립트 실행을 막아둠. 이 한 줄로 풀어주면 됨. **이미 풀려있으면 그냥 다시 메시지 없이 끝나니 신경 안 써도 됨.**

### 2단계 — 아래 명령어 복사해서 붙여넣기

아래 줄을 통째로 복사하세요 (마우스로 드래그):

```powershell
$t="$env:TEMP\lfm-install.ps1"; irm "https://raw.githubusercontent.com/kendrick1028/logic-flow-mapper/main/scripts/install-windows.ps1?r=$(Get-Random)" -OutFile $t; Start-Process powershell -ArgumentList "-NoExit","-ExecutionPolicy","Bypass","-File",$t
```

PowerShell 창에 **우클릭** 하면 자동으로 붙여넣어집니다.

그리고 **Enter** 키 누르세요.

→ **새로운 PowerShell 창**이 자동으로 열리면서 설치가 진행됩니다. 그 새 창에서 7단계가 진행돼요. **무슨 일이 있어도 새 창은 자동으로 안 닫힙니다** (어떤 에러가 나도 메시지 볼 수 있음).

### 3단계 — 자동으로 진행되는 거 지켜보기

화면에 글자가 막 뜨면서 자동으로 다음 것들이 깔립니다:

1. ✅ Node.js (자바스크립트 실행기)
2. ✅ Git (코드 다운로드용)
3. ✅ Claude Code CLI (AI 연결용)
4. ✅ 프로젝트 코드 다운로드
5. ✅ 필요한 부속 파일 설치
6. ✅ 바탕화면에 단축 아이콘 만들기
7. ✅ 프로그램 시작 + 인터넷 창 열기

**아무것도 누르지 말고 5~10분 기다리세요.**

> ⚠️ 중간에 "이거 실행해도 돼?" 같은 빨간 글씨가 뜨면:
>
> PowerShell 창에 아래 명령어 한 번 입력 후 다시 시도:
> ```powershell
> Set-ExecutionPolicy -Scope CurrentUser RemoteSigned
> ```
> "Y" 입력하고 Enter.

### 4단계 — 브라우저에서 Claude 연결

설치 끝나면 자동으로 인터넷 창이 열립니다 (주소: `http://localhost:3000`).

화면 **오른쪽 위**에 주황색 버튼이 있어요:

```
🔌 Claude 연결
```

이 버튼을 **클릭**하세요.

### 5단계 — Anthropic 로그인 (한 번만)

새 탭이 자동으로 열리면서 Anthropic (Claude 회사) 로그인 페이지로 갑니다.

1. 자기 Claude (Max 구독한) 계정으로 로그인
2. **Authorize** 버튼 클릭
3. 끝!

원래 앱 창으로 돌아가면, 주황 버튼이 사라지고 초록색 배지가 뜹니다:

```
🟢 Claude CLI · Max 정액제
```

이게 나오면 **모든 준비 끝!** 이제 자유롭게 쓰면 됩니다.

## 💻 매일 사용하는 방법

### 켤 때

**바탕화면**에서 `Logic Flow Mapper` 아이콘을 **더블클릭**.

- 검정 창이 뜨면서 글자가 막 뜸 (서버 켜지는 중) → **이 창 절대 닫지 마세요**
- 3초 후 인터넷 창이 자동으로 열림
- 이제 사용 가능

### 끌 때

검정 창의 **X 버튼** (빨간색) 클릭. 그러면 자동 종료.

## 🆘 문제가 생기면

### "'irm'은 내부 또는 외부 명령... 아닙니다" 에러

PowerShell 이 아니라 명령 프롬프트(cmd) 에서 실행 중입니다.

해결: 그 창에 `powershell` 입력 후 Enter → 프롬프트가 `PS` 로 시작하게 바뀜 → 그다음 install 명령어 다시 붙여넣기.

또는 cmd 그대로 쓰려면 이 명령어 사용:
```cmd
powershell -ExecutionPolicy Bypass -Command "irm https://raw.githubusercontent.com/kendrick1028/logic-flow-mapper/main/scripts/install-windows.ps1 | iex"
```

### "🔌 Claude 연결" 버튼 눌렀는데 새 탭이 안 열려요

다른 방법으로 로그인:

1. `Windows 키 + X` → PowerShell 열기
2. 아래 명령어 입력:
   ```powershell
   claude setup-token
   ```
3. 화면에 뜨는 URL 을 복사해서 인터넷 브라우저 주소창에 붙여넣기
4. 로그인 → Authorize → 토큰 자동 저장

### 검정 창이 너무 빨리 꺼져요

PowerShell 에서 직접 실행해서 에러 메시지 확인:

```powershell
cd $env:USERPROFILE\logic-flow-mapper
node server.js
```

빨간색 에러 메시지 보이면 그 내용을 알려주세요.

### 브라우저에서 "이 페이지에 연결할 수 없음"

서버가 안 켜진 거예요. 바탕화면 아이콘 다시 더블클릭.

여전히 안 되면:
- `Windows 키 + X` → PowerShell
- `cd $env:USERPROFILE\logic-flow-mapper`
- `node server.js`

### 분석이 너무 느려요

기본은 동시 8개 처리. 더 빠르게 하려면 주소 끝에 `?claudeConcurrency=12` 붙이세요:

```
http://localhost:3000/?claudeConcurrency=12
```

⚠️ 너무 높이면 Anthropic 에서 막을 수 있음. 8~12 사이가 안전.

## 🔄 새로운 버전이 나오면 (업데이트)

만든 사람이 코드를 업데이트했다는 소식 들으면:

### 방법 1 — 다시 설치 명령어 실행

PowerShell 열어서 다시:

```powershell
irm https://raw.githubusercontent.com/kendrick1028/logic-flow-mapper/main/scripts/install-windows.ps1 | iex
```

자동으로 최신 코드 받아옵니다 (5분 정도).

### 방법 2 — git 으로 업데이트만 받기 (빠름)

```powershell
cd $env:USERPROFILE\logic-flow-mapper
git pull
npm install
```
